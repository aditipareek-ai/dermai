# 🩺 DermAI — AI Skin Disease Analyzer

DermAI is an AI-powered skin disease analysis platform that helps users detect and analyze skin conditions using Deep Learning and Computer Vision.

Users can upload skin images and receive instant AI-based predictions along with confidence scores and treatment guidance.

---

## ✨ Features

- 📸 Skin image upload system
- 🤖 AI-powered disease prediction
- 📊 Confidence score analysis
- 🌗 Dark / Light / System theme support
- 👤 Authentication UI
- 📄 PDF-style diagnosis reports
- 📱 Modern responsive UI
- ⚡ Real-time analysis animations
- 🧠 Deep Learning model integration
- 💬 Community remedies section
- 📈 Analysis history tracking

## 🧠 AI Model

The project uses **MobileNetV2** for skin disease classification with image preprocessing and data augmentation techniques.

### Supported Diseases
- Acne
- Eczema
- Psoriasis
- Lupus
- Tinea
- Vasculitis
- Bullous Disease
- Drug Eruption
- Actinic Keratosis
- and more...

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/aditipareek-ai/AI-Skin-Disease-Analyzer.git
```

### Navigate to Project Folder

```bash
cd AI-Skin-Disease-Analyzer
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

## 🚀 How It Works

1. Upload a skin image
2. Image gets processed by the AI model
3. Deep learning model predicts the disease
4. Confidence score is generated
5. Results are displayed instantly
